# Outil Gestion Bibliothèque

## Présentation

Une interface console pour accèder aux fonctionnalités est suffisante.

La solution de persistance peut être au choix :

    * Une base sql locale

    * Des csv

Deux fichiers csv sont fournis, ils comprennent :

    * La liste des livres actuellement au catalogue

    * La liste des abonnés actuels

## Fonctionnalités

### Bibliothèque

* Livres

    * Ajouter un livre au catalogue

    * Supprimer un livre au catalogue

* Abonnés

    * Ajouter un abonné

    * Supprimer un abonné

* Gestion

    * Lister les livres empruntés

### Abonné

* Emprunter un livre

* Rendre un livre
